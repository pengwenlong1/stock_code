# 买卖条件文档

本文档详细说明了 `main.py` 中定义的股票买卖条件，这些条件通过 `judge_buy_id` 和 `judge_sell_id` 进行区分和管理。

## 卖出条件 (`judge_sell_id`)

### `sell_id = 1`：标准卖出逻辑

此 ID 包含一套基于多时间周期和技术指标的综合卖出策略，按以下优先级顺序执行：

1.  **清仓信号**：
    - **条件**：周线出现顶背离，并且 120 分钟 K 线图上出现 MACD 死叉。
    - **操作**：建议清仓。

2.  **减半信号**：
    - **条件**：120 分钟或日线出现顶背离，并且 120 分钟 K 线图上出现 MACD 死叉。
    - **操作**：建议卖出 1/2 仓位。

3.  **阶梯式减仓信号**：
    - **触发条件**：60 分钟 K 线图出现顶背离且 120 分钟图出现 MACD 死叉，**或者** 日线 SAR 指标被跌破。
    - **操作**：根据周线 RSI 指标的强度执行不同比例的卖出：
        - **周线 RSI > 90**：建议卖出全部剩余仓位。
        - **周线 RSI > 85**：建议卖出剩余仓位的 1/2。
        - **周线 RSI > 80**：建议卖出 1/3 仓位。

## 买入条件 (`judge_buy_id`)

系统支持三种不同的买入策略，通过 `judge_buy_id` 进行选择。

### `buy_id = 1`：保守型买入

此策略适用于寻找深度超卖的机会，条件非常严格。

- **条件**：日线 RSI < 20 **并且** 周线 RSI < 25。
- **解读**：仅在短期和长期均处于严重超卖状态时触发，信号稀少但可靠性高。

### `buy_id = 2`：标准型买入

此策略在超卖和技术形态之间取得平衡，是较为常用的买入模型。

- **条件**：满足以下任一即可：
    1.  日线 RSI < 25 **并且** 周线 RSI < 30。
    2.  日线出现底背离 **并且** 120 分钟 K 线图上出现 MACD 金叉。
- **解读**：结合了 RSI 超卖区域和趋势反转的技术形态。

### `buy_id = 3`：指数增强型买入

此策略在标准型基础上，额外增加了对大盘指数的判断，以提高买入的胜率。

- **条件**：满足以下任一即可：
    1.  日线 RSI < 25 **并且** 周线 RSI < 30。
    2.  日线出现底背离 **并且** 120 分钟 K 线图上出现 MACD 金叉。
    3.  **上证指数 (000001)** 或 **创业板指数 (399006)** 的日线 RSI < 25。
- **解读**：当个股出现买入信号，或整个市场处于普遍超卖状态时，均可能触发买入，更适合在市场普跌时捕捉机会。

### `buy_id = 4`：保守型买入 

此策略适用于寻找深度超卖的机会，条件非常严格。 

- **条件**：日线 RSI < 20 **并且** 周线 RSI < 20。 
- **解读**：仅在短期和长期均处于严重超卖状态时触发，信号稀少但可靠性高。
